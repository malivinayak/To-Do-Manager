## About The Project 
To-do Manager is a web development project using HTML, CSS & JavaScript with local storage. Along with front-end To-Do-Manager demonstrate CRUD operation on local storage. [CRUD operation -> Create, Read, Update, and Delete(CRUD)]

To-Do-Manager is a web app used to maintain our day-to-day tasks or list everything that we have to do. This project provides an easy way to add, edit and delete tasks. Added tasks will be stored in the browserâ€™s local storage so, they wonâ€™t be removed on page refresh or tab close.
